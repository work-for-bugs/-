teamWeb目录结构

resources   --存放所有静态资源（就一文件夹）

--------  css         --存放css样式（），包括各界面及如有可能用到的bootstrap统一样式

--------  img         --存放图片资源，各种jpg、png

--------  js          --存放必要的js文件，如axios.js,vue.js等

--------  templates   --存放html文件

--------------    admin       --存放后台管理相关的html文件

--------------    fore        --存放前台（用户使用）相关html文件

--------------    includes    --存放被包含的html文件
